Html Formating

HTML also defines special elements for defining text with a special meaning.

  1. Bold text
  2. Important text
  3. Italic text
  4. Superscript text
  5. Subscript text
  6. Deleted text
  7. Small text
  8. Inserted text
  9. Marked text
  10. Emphasized text
